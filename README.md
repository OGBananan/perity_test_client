This is a repository to help run the websocket client.

Step 1:
Setup the virtual env

```bash
python -m venv env
```


Step 2:
Run the virtual env

```bash
source env/bin/activate
```

Step 3:
Install the dependencies

```bash
pip install -r requirements.txt
```

Step 4: 
Run the file

```bash
python send_wav.py
```
